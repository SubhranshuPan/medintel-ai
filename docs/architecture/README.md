# Architecture

This directory contains architecture documentation, diagrams, and Architecture Decision Records (ADRs) for the MedIntel AI project.

## Structure

```
architecture/
├── diagrams/
└── adr/
```

### diagrams/

Contains Mermaid diagrams used throughout the documentation.

### adr/

Contains Architecture Decision Records documenting significant technical decisions made during the project.

Each ADR explains:

- Context
- Decision
- Alternatives Considered
- Consequences
- Status

The Technical Requirements Document (TRD) references these ADRs where appropriate.
